# 📖 Overview

FakePost is a browser-based social media post generator that allows users to design realistic mock posts for multiple platforms. Customize profile details, captions, images, engagement metrics, comments, and instantly preview the result before exporting it as a PNG.

This project is ideal for UI designers, marketers, content creators, educators, and developers who need realistic social media mockups without editing screenshots manually.

# 🎮 How to Use

1. Choose a social media platform.
2. Upload a profile picture.
3. Enter profile information.
4. Add a caption and image.
5. Customize likes, comments, and views.
6. Preview the post in real time.
7. Download the finished mockup as a PNG.
